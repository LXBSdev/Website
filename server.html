Wilkommen auf LXBS!

Unser Build Server ist jetzt online!

Zusammen mit der neuen Minecraft Version haben wir unseren Build Server veröffentlicht.
Der Server wird von nun an online sein und mit den kommenden Minecraft Versionen mitwachsen.
Auf unserem Build Server könnt ihr eurer Kreativität freien Lauf lassen
und zusammen mit deinen Freunden meisterhafte Kreationen bauen.
Du kannst dem Buildserver nur mit der Version 1.20 beitreten.
Gib einfach lxbs.de ein und schon bist du online.

Tritt unserem Discord bei

Über unseren Server

Unser Server befindet sich noch im Aufbau.
Wir arbeiten jeden Tag an unserem Minecraft Server,
Discord Server und unserer Website.
Unser starkes Team arbeitet so hart wie möglich.
Wir bitten jedoch um Ihr Verständnis.
Im Moment gibt es noch viele Dinge, die überarbeitet werden müssen.
Es könnte einige Bugs geben. Wir bitten euch, euch über Probleme nicht aufzuregen,
sondern sie zu melden. Ihr könnt uns unter support@lxbs.de erreichen, oder auf unserem Discord.
Wir arbeiten auch an einem Feedback Formular für unsere Website.
Wenn ihr Spaß auf unserem Minecraft Server haben wollt,
könnt ihr gerne unserem Discord Server beitreten und einfach auf lxbs.de spielen.

Die Zukunft

In Zukunft wird es eine wunderbare Lobby geben, in der Version 1.20.
Natürlich könnt ihr auch mit allen Versionen darunter spielen.
Von der Lobby aus könnt ihr auf unseren Build Server zugreifen und auf alle anderen Spiele, die geplant sind.
Wir wollen noch nicht zu viel darüber verraten.
Aber wir können schon jetzt sagen, dass es viele klassische Spiele und auch einige Originale geben wird.
Einige werden nur für ausgewählte Versionen verfügbar sein, aber die Klassiker werden für alle Versionen sein.
